
# VLESS XHTTP REALITY + Shadowsocks + Hysteria2 Installer

Одноразовый bash-скрипт для быстрого развёртывания **чистой ноды** (без панели) с поддержкой трёх протоколов:

- **VLESS** + **XHTTP** + **REALITY** (обход цензуры, маскировка под обычный HTTPS-трафик)
- **Shadowsocks 2022** (2022-blake3-aes-256-gcm)
- **Hysteria2** (с обфускацией salamander и маскировкой под сайт)

Скрипт создаёт **ровно одного пользователя** для всех трёх протоколов и сохраняет готовые ссылки для подключения.

### Что делает скрипт за один запуск

1. Устанавливает зависимости
2. Запрашивает домен и email → получает бесплатный SSL от Let's Encrypt
3. Создаёт красивую фейковую страницу логина на 443 порту (TCP) для маскировки Hysteria2
4. Устанавливает свежий **Xray-core** (VLESS + XHTTP + REALITY)
5. Устанавливает свежий **Hysteria2**
6. Генерирует одного пользователя + все необходимые ключи
7. Настраивает и запускает оба сервиса через systemd
8. Сохраняет в `/root/node-keys/credentials.txt`:
   - Имя пользователя
   - UUID / пароли
   - Готовые ссылки для клиентов (VLESS, SS, Hysteria2)

### Требования к серверу

- **ОС**: Ubuntu 20.04 / 22.04 / 24.04 или Debian 11 / 12 (рекомендуется свежая)
- **Права**: root
- **Порты**: открытые 80, 443 (TCP\UDP), 4443 (TCP\UDP) выбранный порт Shadowsocks (обычно 8443 или 2083)
- Минимум **1 ядро / 1 ГБ RAM** (хватит с запасом на 50–100 пользователей)
